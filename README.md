# tantea-legal

Documentos legales de la app **Tantea** (anotador de dominó dominicano).

Servidos vía GitHub Pages en:

- 🌐 https://jairist.github.io/tantea-legal/
- 📜 https://jairist.github.io/tantea-legal/privacy.html (privacidad ES)
- 📜 https://jairist.github.io/tantea-legal/privacy-en.html (privacidad EN)
- 📜 https://jairist.github.io/tantea-legal/terms.html (términos)

## Cómo actualizar

1. Edita el HTML correspondiente.
2. `git add . && git commit -m "Update privacy policy" && git push`.
3. GitHub Pages re-deploya automáticamente en 1-2 minutos.
4. Si cambias material, actualiza la fecha "Última actualización" en el HTML.

## Setup inicial GitHub Pages

Settings → Pages → Source: `Deploy from a branch` → Branch: `main` → Folder: `/ (root)` → Save.
